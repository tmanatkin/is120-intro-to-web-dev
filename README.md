# IS 120 - Introduction to Web Development

This repository contains necessary resources for students to complete homework assignments and access in-class code examples.

## Repository Structure

```
is-120-intro-to-web-dev
├─ homework
│   └─ 00-homework.md
└─ class
    └─ 00-lecture
        ├─ code
        │   └─ index.html
        ├─ class-prep.md
        └─ lecture.md
```

- `homework/` Contains homework assignments for the course.
- `class/` Contains directories for each lecture.
- `class/00-lecture` Contains class prep and lecture material.
- `class/00-lecture/code` Contains code examples and exercises from class.

## Getting Started

To get started, clone this repository to your local machine using the following command:

```bash
git clone <repository-url>
```
